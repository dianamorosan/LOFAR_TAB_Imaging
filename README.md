# LOFAR_dynamic_spectra
 Routines to plot LOFAR tied-array movie using HDF5 dynamic spectra.

 To run:
 
       $ python tiedarray_movie.py
       
 Inputs:
 
       files - list of LOFAR dynamic spectrum HDF5 file
       
Note:  code needs to be in the same folder as the .raw and .h5 files to work!
